# Vercel Landing Page Clone

A **copy/clone attempt** of the Vercel landing page built with vanilla HTML, CSS, and JavaScript.

⚠️ **Note**: This is an educational project recreating Vercel's design. Some parts are incomplete and it's not affiliated with Vercel.

## Features

- Responsive navigation with mobile hamburger menu
- Interactive dropdown menus
- Hero section with embedded video
- Dynamic content switching (AI apps, Web apps, etc.)
- Smooth animations and hover effects
- Mobile-responsive design

## Getting Started

1. Clone this repository
2. Open `index.html` in your browser
3. That's it!

## File Structure

```
├── index.html          # Main HTML file
├── index.css           # Styles
├── index.js            # JavaScript functionality
└── images/             # Framework logos and assets
```

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript (ES6+)
- Intersection Observer API
